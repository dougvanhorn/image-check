# Image Check

Proof of concept to explore using image analytics tools to verify a still image against a video.

Using Python, OpenCV, Ultralytics.

Pretty simple use case, just scratching an itch.

> [!NOTE]
> There are a whole bunch of missing files in the `var` directory.  Namely the videos and still
> images.
>
> You will need at least a video of a bike and a still image of a bike to find this code at all
> useful.
